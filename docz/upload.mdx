# 上传组件
完成对多种文件格式的上传。

## 基本用法

### 点击上传

<PlayGround>
  <Upload 
    action="//jsonplaceholder.typicode.com/posts/"
    listType="picture-card"
    onPreview={file => this.handlePictureCardPreview(file)}
    onRemove={(file, fileList) => this.handleRemove(file, fileList)}
  />
</PlayGround>

### 拖拽上传

<PlayGround>
  <Upload 
    drag
    action="//jsonplaceholder.typicode.com/posts/"
    onPreview={file => this.handlePreview(file)}
    onRemove={(file, fileList) => this.handleRemove(file, fileList)}
    fileList={fileList2}
    listType="picture"
    tip={<div className="el-upload__tip">只能上传jpg/png文件，且不超过500kb</div>}
  />
</PlayGround>

### 照片墙
<PlayGround>
  <Upload

## 属性

属性名|说明|类型|默认值
----|---|---|---
action|请求路径|string|-
headers|请求头设置|object|-
data|请求体数据|object|-
multiple|是否允许多传|bool|false
name|名称|string|-
withCredentials|是否跨域请求|bool|false
showFileList|显示文件列表|bool|true
fileList|文件列表|array|[]
autoUpload|是否自动上传|bool|true
accept|接收jpeg|gif图像|bool|true
drag|是否使用拖拽传输|bool|false
listType|上传形式|string|text
beforeUpload|上传前函数触发|function|-
onRemove|上传内容删除时函数触发|function|-
onPreview|上传文件预览函数触发|function|-
onProgress|上传进度函数触发|function|-
onSuccess|上传成功函数触发|function|-
onError|上传失败函数触发|function|-
onChange|文件上传函数触发|function|-
disabled|是否禁用|bool|false
limit|上传数量限制|number|-
onExceed|超出上传数量函数触发|function|-
httpRequest|上传实现|function|-
