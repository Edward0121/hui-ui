Badge 标记

为按钮和文本提供提示标志。

基本用法

为按钮和文本提供提示标志.

20
<Badge value={'20'}>
  <Button type="primary" size={'small'} onClick={() => {}}>
    icon
  </Button>
</Badge>

自定义最大值

为该组件自定义最大值，如果当前值大于最大值，就返回最大值加+号

19+
<Badge value={20} max={19}>
  <Button type="primary" size={'small'} onClick={() => {}}>
    icon
  </Button>
</Badge>

自定义标记类型

为该组件自定义标记类型，可设置为圆点或普通类型

20
<Badge value={'20'}>
  <Button type="primary" size={'small'} onClick={() => {}}>
    icon
  </Button>
</Badge>
<div style={{ display: 'inline-block', width: '20px' }} />
<Badge value={'20'} isDot={true}>
  <Button type="primary" size={'small'} onClick={() => {}}>
    icon
  </Button>
</Badge>

属性

属性名名称类型默认值
value数值[String, Number]-
max最大值Number-
isDot是否为圆点Booleanfalse
